# 🎵 JavaScript Music Player

A modern and responsive Music Player built using HTML, CSS, and JavaScript.

## ✨ Features

- Play / Pause Music
- Next & Previous Controls
- Song Title & Artist Display
- Progress Bar
- Volume Control
- Playlist Support
- Autoplay Feature
- Responsive Design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

music-player/
│
├── index.html
├── style.css
├── script.js
└── README.md

## 🚀 How To Run

1. Download the project
2. Extract ZIP file
3. Open index.html in browser
